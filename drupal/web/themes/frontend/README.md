# Drupal Frontend Theme

## Run theme (compile / watch)
Install (run outside of vagrant):
```
nvm use
npm install
```

Run watcher

To see the static color and icon preview open the `dist/index.html` in your browser after starting dev.
```
npm run dev
```

Compile assets
```
npm run prod
```

Scaffold components

```
npm run scaffold
```

Run storybook

```
// Needs `npx sb init` first
npm run storybook
```
